1.0.0 - Initial Release

08/05/20

At long last, KaiserBot is joining Kaisercord!

Features:
- Data, games, images, math, moderation, and miscellaneous commands
- Welcome message, autorole, and roles system (Kaisercord only)
- Retrieve info about your favourite groups/soloists
- Do calculus, manipulate images, translate phrases, and define words
- Twitter trends/search data (with graphs!) from 467 different locations
- Up to 20 years of stock market data from 17 different exchanges/indexes

Coming Soon:
- New command category: Music

1.1.0 - Midway Update

25/06/20

No music yet, but a whole host of new commands, fixes, and features.

What's New:
- Hunger Games simulator - play against your friends and/or idols!
- Steam game searcher - check new games, discounts, and ratings
- Book searcher - find relevant books, complete with rating and summary
- Charts command - for finding the most up-to-date k charts information
- Melon command - similar to charts, but Melon exclusively
- Two new Reddit commands: reddit_new and reddit_controversial
- Youtube commands: youtube_top and youtube_search
- Word-of-the-day, courtesy of Merriam Webster
- Urban dictionary searcher
- Uwufier and cloutifier - for transforming text
- Patreon and website commands

What's Updated:
- Fixed division by 0 issues in the eval command
- Changed welcome message to include user avatar
- Fixed cipher 0 issues in cipher command
- Made all Reddit commands Imgur-compatible for better image display
- Added recent-message capabilities to all text transform commands
- And countless other fixes

Coming Soon:
- New command category: Music (really)
- (Potentially) Twitter auto-updates
